
# Function Plotter

A GUI application that plots any given function.

Built with:

 - [Python](https://www.python.org/)
 - [pyside2](https://pypi.org/project/PySide2/)
 - [matplotlib](https://matplotlib.org/)
 - [unittest](https://docs.python.org/3/library/unittest.html)
